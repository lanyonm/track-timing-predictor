U7 Four-state messaging (FR-010)

Messages appear once above the session list using role="status" for screen reader live region announcements. Four states: (a) success, (b) no-match warning (only when start lists exist), (c) missing start lists info, (d) no-data (all start lists missing — suppresses misleading "no matches").
